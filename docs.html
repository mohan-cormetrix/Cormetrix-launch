Introduction

Welcome to the CorMetrix documentation. This comprehensive guide will help you get started with our platform and products, providing detailed information on installation, configuration, and usage.

Note: This documentation is regularly updated. Make sure you're viewing the latest version.

About CorMetrix

CorMetrix is a healthcare and legal technology company that builds AI-powered solutions to help professionals make better decisions. Our platform turns complex data into structured intelligence that improves with every case.

Quick Start Guide

Get up and running with CorMetrix in minutes:

  1. Sign up for a CorMetrix account
  2. Choose your product (VerixAI or AtriAI)
  3. Complete the onboarding process
  4. Upload your first case or patient data
  5. Start using the platform

System Requirements

CorMetrix is a cloud-based platform that works in any modern web browser. For optimal performance, we recommend:

  • Chrome, Firefox, Safari, or Edge (latest versions)
  • Minimum 4GB RAM
  • High-speed internet connection
  • Screen resolution of 1280x800 or higher

Installation

As a cloud-based solution, CorMetrix requires no installation on your local systems. Simply access the platform through your web browser at app.cormetrix.com.

For Enterprise Customers

Enterprise customers with on-premises requirements should contact our implementation team for detailed deployment instructions.

Platform Overview

The CorMetrix platform is built on a secure, scalable architecture designed to handle complex healthcare and legal data. Our platform consists of several key components:

  • Data Ingestion Layer - Securely imports and processes documents, images, and structured data
  • AI Processing Engine - Analyzes and extracts meaningful information from raw data
  • User Interface - Intuitive web-based interface for interacting with the platform
  • API Layer - Secure REST APIs for integration with external systems

Security Features

Security is built into every layer of our platform:

  • End-to-end encryption for all data in transit and at rest
  • Role-based access control with fine-grained permissions
  • Comprehensive audit logging of all system activities
  • Regular security assessments and penetration testing

API Reference

Our REST API allows you to integrate CorMetrix functionality into your existing systems. All API endpoints are secured using OAuth 2.0.

Authentication

To authenticate with the API, you need to obtain an access token:

POST /api/v1/auth/token
Content-Type: application/json

{
  "client_id": "your_client_id",
  "client_secret": "your_client_secret",
  "grant_type": "client_credentials"
}

Example API Call

Here's an example of how to retrieve a list of cases:

GET /api/v1/cases
Authorization: Bearer your_access_token
Note: For detailed API documentation, please refer to our API Documentation Portal.

Integrations

CorMetrix integrates with a wide range of healthcare and legal systems:

  • EHR Systems - Epic, Cerner, Allscripts, and more
  • Imaging Systems - PACS, VNA, and other DICOM-compatible systems
  • Document Management - SharePoint, Box, Dropbox, Google Drive
  • Legal Case Management - Clio, PracticePanther, and custom systems

For custom integration requirements, please contact our implementation team.

VerixAI Documentation

VerixAI is our medico-legal solution designed to help legal teams review and analyze medical records more efficiently.

Key Features

  • Unified Cloud-Based Platform - Consolidates data and imaging into one unified platform
  • Automated Chronologies - Generates accurate medical chronologies within minutes
  • Natural Language Queries - Ask questions about cases in plain English

Getting Started with VerixAI

To get started with VerixAI, follow these steps:

  1. Log in to your CorMetrix account
  2. Select VerixAI from the product dashboard
  3. Create a new case and upload documents
  4. Wait for the AI to process the documents
  5. Start exploring the case data

AtriAI Documentation

AtriAI is our healthcare solution focused on left atrial appendage occlusion (LAAO) procedures.

Key Features

  • Comprehensive Workflow - Manages the entire LAAO procedure workflow
  • AI-Powered Analysis - Analyzes imaging data to assist with device selection
  • Automated Documentation - Generates procedure reports and documentation

Getting Started with AtriAI

To get started with AtriAI, follow these steps:

  1. Log in to your CorMetrix account
  2. Select AtriAI from the product dashboard
  3. Create a new patient record
  4. Upload imaging studies and patient data
  5. Start using the workflow tools
Important: AtriAI is designed to assist healthcare professionals but does not replace clinical judgment. Always verify all information and recommendations.
